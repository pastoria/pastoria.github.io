    sudo apt-get install git

  • 将安装包解压到指定目录

    sudo tar xJvf node-v12.17.0-linux-x64.tar.xz -C /usr/local/bin/

  • 确保”/usr/local/bin/“在PATH路径中

  • 写文章生成网页本地预览

  • 配置_config.yml部署参数
