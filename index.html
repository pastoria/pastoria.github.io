Ubuntu搭建Hexo个人博客

发表于 2020-05-27 更新于 2020-05-29

  • 安装git

    sudo apt-get install git

  • Node.js官网下载Linux Binaries (x64)安装包

    node-v12.17.0-linux-x64.tar.xz

  • 将安装包解压到指定目录

    sudo tar xJvf node-v12.17.0-linux-x64.tar.xz -C /usr/local/bin/

  • 创建软链接

  • 确保”/usr/local/bin/“在PATH路径中

  • 在用户目录中查看node及npm版本

  • 安装Hexo

    sudo npm install hexo-cli -g

  • 创建并初始化blog目录

  • 写文章生成网页本地预览

    > INFO Hexo is running at http://localhost:4000 . Press Ctrl+C to stop.

  • 安装hexo部署插件

  • 配置_config.yml部署参数

  • 部署到GitHub Pages

    hexo d

Ubuntu环境安装vim支持系统剪贴板

发表于 2020-05-27 更新于 2020-05-30

  • Ubuntu自带vim不支持复制粘贴系统剪贴板，需要安装vim插件

    sudo apt-get install vim-gtk vim-scripts vim-gnome

  • 查看安装好的vim插件

    dpkg -l vim*

  • 查看vim是否支持系统剪贴板

    vim --version | grep clipboard

    xterm_clipboard前面+表示支持系统剪贴板，-表示不支持

  • 在vim中使用系统剪贴板复制粘贴文字

    • 复制 "+y
    • 粘贴 "+p

Ubuntu解压压缩文件命令

发表于 2020-05-27

  • 解压tar.gz

    sudo tar xzvf openvpn-2.4.8.tar.gz -C /usr/local/bin/

  • 解压tar.xz

    sudo tar xJvf node-v12.17.0-linux-x64.tar.xz -C /usr/local/bin/

  • 解压tar.bz2

    sudo tar xjvf openvpn-2.4.8.tar.bz2 -C /usr/local/bin/

  • 解压zip

    unzip -d ~/Documents git-recipes-master.zip
