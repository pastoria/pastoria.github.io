Ubuntu环境安装vim支持系统剪贴板

  • Ubuntu自带vim不支持复制粘贴系统剪贴板，需要安装vim插件

    sudo apt-get install vim-gtk vim-scripts vim-gnome

  • 查看安装好的vim插件

    dpkg -l vim*

  • 查看vim是否支持系统剪贴板

    vim --version | grep clipboard

    xterm_clipboard前面+表示支持系统剪贴板，-表示不支持

  • 在vim中使用系统剪贴板复制粘贴文字

    • 复制 "+y
    • 粘贴 "+p

Ubuntu解压压缩文件命令

  • 解压tar.gz

    sudo tar xzvf openvpn-2.4.8.tar.gz -C /usr/local/bin/

  • 解压tar.xz

    sudo tar xJvf node-v12.17.0-linux-x64.tar.xz -C /usr/local/bin/

  • 解压tar.bz2

    sudo tar xjvf openvpn-2.4.8.tar.bz2 -C /usr/local/bin/

  • 解压zip

    unzip -d ~/Documents git-recipes-master.zip
